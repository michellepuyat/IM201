White Space and Clean Design

Google

www.google.com

Google's web page is a perfect example for using white space and clean design principle. The website looks clean which makes it easy to focus on the main objective of the website which is to search.

PARC: Contrast

Spotify

https://www.spotify.com/us/

Spotify's web page used contrasting colors which made text readable. Call-to-action button stood out because of the contrast.

Fritt's Law

Zalora

https://www.zalora.com.ph/women/

Zalora's website uses banners to promote sales and discounts to certain products. This directs user to discounted products easier instead of browsing to each menu to look for discounts.
